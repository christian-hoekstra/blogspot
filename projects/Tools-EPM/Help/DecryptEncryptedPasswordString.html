Decrypt Encrypted Password String

Displays the clear text password from a encrypted password string by using the security key stored in Shared Services. Optionally you can use enclose the encrypted password string by single quotes, in case the string contains special characters.


D:\> cryptTool.cmd|sh -d <encrypted password string>




Decrypt Password String using a specified Security Key

Displays the clear text password from a encrypted password string by using the security key specified.


D:\> cryptTool.cmd|sh -d <encrypted password string> -k <security key>
