Encrypt Password String

Creates a encryted password string based on the security key stored in Shared Services.


D:\> cryptTool.cmd|sh -e <password string>




Encrypt Password String using a specified Security Key

Creates a encryted password string based on the security key specified.

D:\> cryptTool.cmd|sh -e <password string> -k <security key>
