manageAdmin.cmd|sh

Manages the factory default [admin] user account of Oracle EPM. Allows you to print its given user-name, and change (reset) its password.


Features

  • No credentials are required in order to use this utility.
  • New password effective immidiatly, no restart of services required.


Limitations

  • Must be executed from a system that has Foundation Shared Services installed.
  • Factory build-in admin user only, no other users can be reset for obvious security reasons.
  • Provisioning cannot be altered.
