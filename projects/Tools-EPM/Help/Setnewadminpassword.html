Set new password for the [admin] user

In case the password has been lost for the factory [admin] account, it can be reset with this command. No credentials needed. You cannot retrieve the original password, only set a new one.

There are two modes: Provide the new password as argument, or ommit the password and the tool will ask for the new password.


The name between brackets [] shows the actual configured admin username (as it would show when using -i parameter). In case of the example below the name is "admin". When password has succesfully been changed, the [ OK ] message will appear. If something whent wrong, the message [ FAILED ] will appear. In case of failure, please check the tools-epm.log for details.


  • Ommit new Password

For security reasons, the password is hidden while entering.


D:\> manageAdmin.cmd|sh -s




  • Pass new password as argument

The new password is provided as an argument in plain text (not encrypted). Please be aware the password is visible to others while the shell window is open.


D:\> manageAdmin.cmd|sh -s <password>
