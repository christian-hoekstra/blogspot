Specify report export format

Default output format is HTML. By adding parameter -e, --exportFormat to the parameters -aut or -oh, you can specify to export to XML instead. Only HTML or XML is supported.


D:\> manageOPatch.cmd|sh -oh <\path\to\Oracle\home> -e xml




D:\> manageOPatch.cmd|sh -aut -e xml
