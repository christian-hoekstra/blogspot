System Requirements

Introduction ››

Tools-EPM required system configuration includes:

  • Windows 2008 or Windows 2012 including R2 editions
  • Java JDK 1.6, packaged with EPM
  • Oracle EPM (Enterprise Performance Management) 11.1.2.3 or 11.1.2.4 installed
  • Some tools are dependent on Shared Services (installed)
  • Oracle Common and Weblogic must be available in the Middleware home folder


Limitations:

  • Spaces in Tools-EPM installation path is not supported
  • EPM 11.1.2.1 and older is not supported for some utilities


Test on/with:

  • Windows 2012 R2
  • Windows 2008 R2
  • Oracle EPM 11.1.2.3
  • Oracle EPM 11.1.2.4
